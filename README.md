**P53 activation in acute myeloid leukaemia (RNA-seq)**: We have compared AML stem cells from untreated patients to those treated with an MDM2 inhibitor, 
a BET inhibitor and the combination of the two treatments. Both of these treatments aim to switch P53back on, so that it can prevent cancer cells from replicating.
The aim was to determinewhat genes and pathways are affected by each drug and the combination. Does thecombination do anything different to each drug on its own
